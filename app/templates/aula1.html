Aula 1 - Introdução ao Flask

Nesta primeira aula, quero compartilhar minha experiência no mundo do desenvolvimento web com Python usando o Flask. Começamos pelo básico, instalando o Flask e criando uma aplicação web simples. Esta aula foi o ponto de partida para criar minhas próprias aplicações web com Flask.
